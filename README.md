# MKSnapShotter-With-MKPolylines-UITableView-Example

Simple project for example how to create a MKSnapShotter with MKPolylines (path on the map) and cached it. So, it make the performance scroll higher.

##### Requirements
<b>Xcode:</b> >= 8.3.2<br>
<b>iOS:</b> >= 8.0<br>
